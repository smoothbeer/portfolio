Overview

A web-based IPTV player built so I could watch football in VR. Couldn't find one that worked, so I built one with Claude Code. Load an M3U playlist, pick a channel, watch. It just works.

View project →
Interfaces

Three interface modes, each built for a different situation.

Mode Description
Normal Standard player. Channel list, search, video. Nothing fancy, just works.
TeeFax A ceefax-inspired interface designed to run on slow browsers or bad internet. Stripped back to text and numbers so you can still find what you need when the connection is struggling.
Work Mode Hides the player inside a mock desktop with a fake spreadsheet in front of it. For watching TV when you should be working.
